# 🔍 멀티모달 한국어 가짜뉴스 탐지 연구

> 국민대학교 경영정보학부 ITIS 학회 | 2기 학부생 연구 프로젝트

## 📌 연구 개요

한국어 정치뉴스 도메인에서 이미지+텍스트 기반 멀티모달 가짜뉴스 탐지 모델을 개발하고,
GenAI로 생성한 가짜 이미지·왜곡 텍스트 기반 데이터셋을 구축한다.

## 🔧 조작 수준 기준표

### 텍스트 LV
| Level | 정의 | 예시 |
|---|---|---|
| LV1 | 단어/어조만 변경 | "발표" → "논란의 발표" |
| LV2 | 일부 정보 생략, 맥락 왜곡 | 긍정 발언 삭제 |
| LV3 | 감정/의도 추가 | "사실상 실패 인정" |
| LV4 | 허위 출처/통계 삽입 | "관계자에 따르면" |
| LV5 | 존재하지 않는 사건 생성 | 없는 비리 날조 |

### 이미지 LV
| Level | 정의 | 예시 |
|---|---|---|
| LV1 | 색감/밝기/크롭 변경 | 어둡게 보정 |
| LV2 | 특정 장면만 노출 | 무표정 장면만 사용 |
| LV3 | 자막/화살표 추가 | "당황한 표정" 자막 |
| LV4 | 가짜 문서/객체 합성 | 가짜 공문서 삽입 |
| LV5 | AI 생성/딥페이크 | 존재하지 않는 현장 |


## 📅 연구 일정

- Phase 1 (6월): 연구 설계 및 데이터셋 구조 확정
- Phase 2 (7~8월): 데이터셋 구축
- Phase 3 (9~10월): 탐지 모델 학습 및 평가
